WanderLust – Airbnb Clone

WanderLust is a full-stack Airbnb-inspired web application where users can browse, create, edit, and manage property listings. The project includes secure user authentication, image uploads, reviews, and a responsive user interface.

🚀 Live Demo

🌐 https://wanderlust-fben.onrender.com

📂 GitHub Repository

💻 https://github.com/omkarpawde/wanderlust-new

✨ Features
🔐 User authentication (Sign Up, Log In, Log Out)
🏠 Create, edit, and delete property listings
📸 Upload listing images using Cloudinary
⭐ Add and delete reviews with ratings
👤 Listing ownership and authorization
📱 Responsive design for desktop, tablet, and mobile
💾 MongoDB Atlas database integration
🔔 Flash messages for user actions
🛡️ Session management with Passport.js
